# b) 2. Fixed XOR.

After reading [this](https://medium.com/analytics-vidhya/crypto-basics-fixed-xor-implementation-python-9cfba54f4661) article I had a relatively good understanding on what to do, actually this kind of spoiled the whole answer for me (but thats okay)

The key idea from the article is that instead of XORing bytes directly, you convert hex → integer → binary string, pad them to equal length with zfill(), XOR bit by bit, then convert back from binary → integer → hex. 

So the conversion happens like this 
* bin_1 = bin(int(hex_1, 16))[2:] same for the other variable,
we make it equal lenght by checking the max len of both the bin variables, we XOR bit by bit with a for loop 
* result = [int(bit1) ^ int(bit2) for bit1, bit2 in zip(bin_1, bin_2)]
Then we convert binary back to hex and return the output
* final_output = hex(int(string_result, 2))[2:]
